상어 아일랜드 게임 
• 배열 등의 변수 문법을 활용하여 각종 요소 표현
• 일상적인 동작 규칙을 수식 및 코드로 표현
• 함수 모듈화 및 프로그램 구조화를 통해 전체를 세분화
• 단계적으로 구현하는 과정에 대한 실습

게임 규칙
• 입력된 player 이름으로 표기 (player 수는 고정)
• 주사위를 굴리면서 앞으로 이동
• 도착한 칸에 coin이 있으면 습득
• 상어가 쫓아오기 전에 끝까지 이동
• 살아있는 player 중 coin이 가장 많은 player가 승리

게임 흐름
• 각종 변수 초기화 및 플레이어 이름 입력
• 플레이어 별로 턴을 돌면서 반복 동작 수행
• 게임 상태 출력
• 주사위를 굴리고 이동 (coin 습득)
• 적절한 시점에 상어 이동
• 게임 종료 후 출력
